# SerpiumVPN 1.0.10

## UX автоподбора стратегий

- Добавлен видимый блок под кнопкой «Способ 2».
- Во время автоподбора показываются текущая стратегия, шаг проверки и прогресс.
- При слабом соединении программа больше не должна выглядеть зависшей: пользователь видит, что идёт перебор стратегий.
- После успешного подбора показывается найденная стратегия.
- При остановке подбор корректно помечается как отменённый.

## Релиз через GitHub Releases

Для публикации OTA-обновления через текущий механизм GitHub Releases:

```powershell
.\release.ps1 -Version 1.0.10 -SelfContained -PublishGitHub
```

Скрипт должен создать и загрузить в релиз `v1.0.10` два asset-файла:

- `SerpiumVPN-1.0.10.zip`
- `update.json`

Именно `update.json` сообщает установленной программе новую версию, ссылку на zip и sha256.
