SerpiumVPN OTA updater fix

Что меняется:
- updater запускается без повторного UAC/runas;
- аргументы передаются через ProcessStartInfo.ArgumentList;
- Windows-пути больше не экранируются вручную;
- добавлена проверка, что updater не завершился сразу после запуска;
- сохраняются bootstrap/error логи в %TEMP%.

Установка:
1. Распаковать архив в корень проекта с заменой.
2. Убедиться, что Program.cs попал в SerpiumUpdater\Program.cs.
3. Выполнить dotnet build.
4. Собрать новый установщик/релиз.

Важно:
Этот фикс должен попасть в установленную основную программу хотя бы один раз через новый установщик.
Только после этого следующие OTA-обновления смогут применяться автоматически.
